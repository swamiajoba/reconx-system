# Student Guides — ReconX Advanced (Intermediate-Hybrid) Track

> Deutsche Bank — TDI 2026 Graduate Technical Training Programme

This folder is what **you** (the grad) read each day. One sub-folder per day,
each with a `README.md` that contains:

1. **Day overview** — what you'll learn, what you'll ship by EOD.
2. **Module plan** — AM theory + Percipio labs, PM exercises.
3. **Exercises** — *Day N · Ex N.M* with acceptance criteria + hints (NOT solutions).
4. **Stretch goal + mini-challenge** — for fast-finishers.
5. **End-of-day checklist** — what should be green before standup tomorrow.
